02 · 引导设置

首次启动 4 步。

独立窗口，居中模态。 每步硬门槛：凭据 → 麦克风 → 辅助功能 → 试用。第 3 步授权后 app 自动重启，从第 4 步恢复。

欢迎使用 whoosh
第 01 步 · 凭据

连接到 豆包·Seed ASR。

在火山引擎控制台「语音识别」资源页找到对应字段。Access Token 仅本机保存（macOS Keychain / Windows DPAPI），不会上传。

连通性：已验证 · 服务端响应 142ms
OK
1 / 4
第 02 步 · 麦克风

允许 whoosh 访问麦克风。

录音数据仅作为 ASR 流的输入，不会本地缓存（debug 模式除外，需手动开启）、不会上传到 Doubao 之外的任何服务器。

麦克风访问 系统弹窗将出现一次，点击「允许」完成授权。
必需
2 / 4
第 03 步 · 辅助功能（仅 macOS）

允许全局快捷键与键盘注入。

macOS 需要在「系统设置 → 隐私与安全 → 辅助功能」勾选 whoosh，用于：监听全局右 Option 触发键、模拟 ⌘V 把识别文本粘贴到当前 app。授权后 app 会自动重启一次。

辅助功能访问 点击下方按钮跳转到「系统设置 → 隐私与安全 → 辅助功能」。
必需
授权完成后 whoosh 会自动重启以加载键盘监听 hook。当前进度已保存，将自动恢复到第 4 步。
3 / 4
第 04 步 · 试用

说一句，或敲一句。

按住右 Option 说话试试，或直接键盘输入。任意非空文本即可完成首次设置。
